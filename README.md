# coda-command-line

coda is a command-line tool for Coda 2.

## Usage

    $ coda file1.m file2.m file3.m

    $ coda folder/*.m

    $ coda --new-window file1.m

## Installation

1. Open coda.xcodeproj and push Build.
2. That's it
